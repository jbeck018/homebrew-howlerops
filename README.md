# Homebrew Howlerops

Homebrew tap for [HowlerOps](https://github.com/jbeck018/howlerops) - A powerful SQL client and operations platform.

## Installation

### Desktop App (GUI)

```bash
brew tap jbeck018/howlerops
brew install --cask howlerops
```

### Backend CLI (Coming Soon)

```bash
brew tap jbeck018/howlerops
brew install sql-studio-backend
```

## What's Included

- **HowlerOps Desktop**: Full-featured SQL client with AI capabilities (macOS Universal Binary)
- **SQL Studio Backend**: Self-hosted backend service for team deployments (Coming Soon)

## Requirements

- macOS 11.0 (Big Sur) or later
- Apple Silicon (M1/M2/M3) or Intel processors supported
